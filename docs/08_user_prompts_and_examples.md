# 用户提示词与回答示例

这份文档用于设计小程序里的快捷问题、AI 提示词、测试用例和运营内容。

## 1. 快捷入口文案

首页快捷入口：

- 今天该怎么喂？
- 帮我判断症状
- 生成换粮计划
- 便便是否正常？
- 疫苗驱虫提醒
- 去医院前整理病情
- 解释化验单
- 挑选主粮建议

症状快捷入口：

- 呕吐
- 腹泻
- 不吃饭
- 咳嗽/打喷嚏
- 掉毛/皮肤痒
- 耳朵脏/甩头
- 眼睛红/流泪
- 尿频/尿不出
- 误食异物
- 精神很差

餐食快捷入口：

- 幼猫怎么喂？
- 幼犬怎么喂？
- 绝育后怎么控制体重？
- 老年猫狗怎么吃？
- 肠胃敏感怎么换粮？
- 主粮和湿粮怎么搭配？
- 零食一天能吃多少？
- 自制饭要注意什么？

## 2. 用户输入模板

### 症状判断模板

```text
我的宠物是【猫/狗】，【年龄】岁，【体重】kg，【是否绝育】。
它从【时间】开始出现【症状】，目前【精神状态】、【食欲】、【喝水】、【排便排尿】。
最近是否换粮/吃药/误食/外出：【说明】。
请帮我判断紧急程度，以及现在应该做什么。
```

### 餐食搭配模板

```text
我的宠物是【猫/狗】，【年龄】岁，【体重】kg，【是否绝育】，
当前吃【主粮/湿粮/自制/零食】，【活动量】。
```


## 3. AI 回答示例：餐食搭配

用户：

```text
我的猫 2 岁，4.8kg，已绝育，室内活动少，有点胖。现在吃干粮，偶尔吃冻干。怎么搭配餐食？
```

建议回答：

```text
先按体重管理处理：它不是必须减很快，但需要控制热量和零食比例。

建议：
1. 主粮仍作为基础，按包装建议量的低值开始，不要自由采食。
2. 湿粮可以替代一部分干粮，帮助增加饮水和饱腹感。
3. 冻干按零食处理，每日热量最好不超过总热量的 10%。
4. 每周称重 1 次，目标是缓慢下降，不建议突然大幅减量。
5. 如果出现不吃、呕吐、腹泻、精神变差，要停止调整并咨询兽医。

如果你愿意，我可以根据它当前粮的热量和每天喂多少，帮你算一个更具体的 7 天调整计划。
```

## 4. AI 回答示例：黄色症状

用户：

```text
狗狗 3 岁，10kg，今天拉稀 3 次，精神还可以，吃得少一点，没有吐。
```

建议回答：

```text
风险等级：黄色，建议密切观察，并在 24-48 小时内联系宠物医院；如果加重，需要更早就医。

这不等于诊断。拉稀可能和换粮、吃到不合适的东西、寄生虫、胃肠炎等有关。现在最重要的是观察精神、饮水、呕吐、便血和次数变化。

现在可以做：
1. 暂停零食和新食物，保持清洁饮水。
2. 记录便便次数、颜色、是否带血或黏液。
3. 不要自行喂人用止泻药或抗生素。
4. 如果有近期换粮，先停止继续加量。

立即就医信号：
便血、黑便、持续呕吐、精神明显变差、不喝水、腹痛、发热、幼犬/老年犬、腹泻超过 24 小时仍无改善。

就医前可准备：便便照片、最近食物变化、驱虫时间、疫苗情况、是否误食。
```

## 5. AI 回答示例：红色急症

用户：

```text
公猫一直蹲猫砂盆，尿不出来，还叫。
```

建议回答：

```text
风险等级：红色。请现在就联系附近宠物医院或急诊，公猫尿不出来可能是危及生命的急症。

不要在家等待观察，也不要自行喂药或强行按摩腹部。请尽快带猫去医院，并告诉医生：
- 公猫，出现频繁蹲猫砂盆但尿不出。
- 是否有叫声、舔尿道口、精神差、呕吐。
- 最近饮水、排尿、食物变化。
- 症状从什么时候开始。

这不是诊断，但这种情况需要线下检查和处理，延误可能造成严重后果。
```

## 6. 越权问题拦截

用户：

```text
狗狗发烧了，阿莫西林喂多少？
```

建议回答：

```text
我不能为宠物开具抗生素剂量或处方。狗狗发烧背后可能有感染、炎症、中暑、免疫问题等多种原因，用错药会掩盖病情或造成风险。

建议先确认体温测量方式和具体数值，并尽快联系宠物医院。就医前请记录：体温、精神、食欲、饮水、是否呕吐腹泻、是否咳嗽、近期是否外出或受伤。

不要自行使用人用药或抗生素。
```
